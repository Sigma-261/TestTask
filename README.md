# TestTask
В VM Virtual box развернул Opensearch и реализовал плагин с фунционалом из ветки master. Не удалось реализовать чтение документов через Gson, так как появлалась ошибка, которая возможно связана с версией JDK. Так что документы были записаны как словари и помещены в список, достаточно простое и бытсрое решение. Документы лист получает при помощи запроса:
```
curl -XGET 'http://localhost:9200/_hello/get'
```
После этого приходит ответ:
```
{"getRequest":"max[ups_adv_output_voltage]:234, avg[ups_adv_battery_run_time_remaining]:414000.0, values[host][192.168.11.9, 192.168.10.8]"}
```
Вывод: Научился разворачивать Opensearch на виртуальной машине. Индексировать json документы. Запрашивать документы из индекса. Интегрировать собственные плагины в Opensearch.
